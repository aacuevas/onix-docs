HeartbeatDevice¶

A Bonsai Source that wraps a Heartbeat device.

Heartbeat.bonsai

Inputs¶

None

Outputs¶

A single HearbeatFrame every time one is produced by hardware. This type is a direct wrapper around the Device To Host Data Frame specified on the Heartbeat device datasheet.

Configuration¶

Configuration of Heartbeat is performed using its property pane. Parameter explanations are provided there.
